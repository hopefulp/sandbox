##### directory manual
cal_chg.sh              calculate molecular charge in .mol2 file or .top file
                        part of "grmx_start.py"
fline_extract.py        extract lines regularly in a file
grmx_anal.py            analysis of gromacs results such as .trr or .edr
grmx_start.py           run gromacs from start of qchem input file
histo_f.py              plot 2d histogram of a file
mol2pdb.pl              written by CW Kim, convert Q-Chem input file to pdb format
mp_line_f.py            matplotlib of 2D line
top_gaff_add_ion.sh     add a line of FF to .top file
top_gaff_add_water.sh   add two lines of FF to .top file
